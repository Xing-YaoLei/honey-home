## 智能社区云平台

> 制作一个智能社区的系统，主要服务对象为小区业主，由物业公司或者第三方单位运营。只对当前小区提供服务。包含有：小区论坛、小区新闻发布、小区活动(报名)、小区购物(社区购物)

## 前台

### 公告

小区的公告信息，比如：停水停电公告，集体维修公告等等

- 公告列表

  分页的方式，按时间倒叙获取公告列表。列表中不包含详情数据

- 公告详情

  根据id获取公告的详情信息

### 资讯

小区的一些新闻趣事

- 资讯分类数据

  资讯的类别信息，如果需要可以在列表页制作一个分类切换的功能

- 资讯列表

  资讯列表数据，如果有传递参数分类id，那么获取的是指定分类的资讯数据

- 资讯详情

  根据id获取资讯详情

### 活动

小区组织的一些社区活动，比如：自驾、旅游、比赛项目等，可以开启报名功能。报名需提供姓名、手机号、参与人数等

- 活动分类数据

  活动类别数据

- 活动列表

  活动列表数据，如果传递的有参数分类id，那么返回指定分类的活动。否则返回所有

- 活动详情

  根据id获取活动详情数据

- 活动报名

  活动报名，用户可以填写个人报名信息进行报名活动。必须登录之后才能报名活动

### 论坛

- 板块列表

  论坛的板块信息

- 帖子列表

  如果是自己发布的帖子，提供修改和删除功能

  获取审核过的帖子列表，如果传递板块参数，那么获取的是当前分类的帖子

- 帖子详情

  获取帖子详情

- 发布帖子

  发布新帖子

- 评论列表

  根据帖子id获取帖子的评论列表

- 发布评论

  发布评论

- 热门帖子

  获取浏览量最多的帖子，可以传递一个返回的数量作为参数

- 最新帖子

  获取最新发布的帖子，可以传递一个返回的数量作为参数

### 商城

#### 商品部分

- 商品分类

  商品类别数据

- 商品列表

  商品列表，如果有分类根据分类id获取指定分类的商品

- 商品详情

  根据id获取商品详情

- 发布评论

#### 购物车

- 新增

  加入购物车

- 修改

  修改购物车中商品数量

- 删除

  删除购物车数据

#### 订单

- 购买

  提交订单

- 获取订单列表(根据状态(支付状态、订单状态))

  获取订单列表，可以传递参数 支付状态(0/1)，订单状态(0/1)

- 获取订单详情

  根据订单id获取订单详情

### 个人中心

- 注册

- 修改密码
- 签到

## 数据库设计

- 用户表(users)

  ```
  user_name	用户名
  nick_name	昵称
  age			年龄
  birthday	生日
  address		详细信息
  area		楼栋号
  password	密码
  is_locked	是否锁定
  score		积分
  avatar 头像
  gender	性别
  ```

- 轮播图分类(banner_categories)

  ```
  name		名字
  desc		描述
  ```

- 轮播图(banners)

  ```
  name				名字
  link				链接
  url					资源路径
  desc				描述
  category		分类id
  ```

- 资讯分类(article_categories)

  ```
  name				名字
  cover_image	主图
  desc				描述
  parent  父级id
  ```

- 资讯(articles)

  ```
  name				名字
  cover_image	主图
  desc				描述
  conten 			详情
  category		分类id
  views				浏览次数
  ```

- 公告(notices)

  ```
  name		名字
  cover_image	主图
  desc		描述
  conten 		详情
  views		浏览次数
  ```

- 论坛版块(forums)

  ```
  name		名字
  cover_image	主图
  desc		描述
  parent  父级id
  ```

- 论坛(posts)

  ```
  name				名字
  cover_image	主图
  desc			描述
  conten 		详情
  forum			版块id
  views			浏览次数
  user			用户id
  ```

- 评论(comments)

  ```
  conten 		详情
  post			帖子id
  views			浏览次数
  user			用户id
  ```
  
- 商品分类(product_categories)

  ```
  name		名字
  cover_image	主图
  desc		描述
  parent  父级id
  ```

- 商品(products)

  ```
  name				名字
  cover_image	主图
  desc				描述
  conten 		详情
  category	商品分类id
  views		浏览次数
  price		价格
  amount	库存
  on_sale		是否在售
  ```

- 商品评论(product_comments)

  ```
  user_id		用户id
  product_id	商品id
  content		评论内容
  cover_image	主图
  ```

  

- 购物车(carts)

  ```
  user			用户id
  product		商品id
  price			价格
  amount		数量
  ```

- 订单(orders)

  ```
  user			用户id
  address		地址
  pay_status	支付状态(0未支付，1已支付)
  order_status	订单状态(0未完成，1已完成)
  price		金额
  ```

- 订单详情(order_details)

  ```
  order				订单id
  product			商品id
  price				商品价格
  amount			购买数量
  ```

- 积分记录(score_logs)

  ```
  user		用户id
  score		积分
  remarks		备注
  ```

- 管理员信息表

  ```
  user_name	用户名
  password	密码
  nick_name	昵称
  gender		性别
  avatar		头像
  ```
